# Pt. Prashant Dubey — Astrology Consultation Website

A modern, responsive astrology consultation landing page built with HTML, CSS, and JavaScript.  
Designed for professional astrologers, spiritual consultants, and wellness practitioners offering online and in-person services.

---

## ✨ Features

- Responsive navigation with mobile menu
- Hero section with CTA buttons
- About section with credentials and experience
- Services showcase with pricing
- Interactive booking section with embedded Typeform
- Weekly horoscope section
- Testimonials section
- Blog preview section
- Animated statistics counters
- Smooth scrolling navigation
- Mobile-friendly design
- Elegant spiritual / luxury UI aesthetic
